INSTRUÇÕES DE INSTALAÇÃO NO CPANEL (Node.js)
==============================================

Este sistema é feito em Node.js. Para rodar no cPanel, sua hospedagem precisa ter a opção "Setup Node.js App" (ou "Configurar Aplicação Node.js").

PASSO A PASSO:

1. Upload dos Arquivos
   - Faça o upload do arquivo 'rastreio-encomendas.zip' para a pasta desejada no Gerenciador de Arquivos do cPanel (ex: public_html ou uma subpasta).
   - Extraia (Unzip) o arquivo.

2. Configurar o Node.js no cPanel
   - Procure por "Setup Node.js App" ou "Node.js" no painel do cPanel.
   - Clique em "Create Application" (Criar Aplicação).
   - Preencha os campos:
     * Node.js Version: Escolha a versão 18 ou superior.
     * Application Mode: Production.
     * Application Root: O caminho da pasta onde você extraiu os arquivos (ex: public_html/rastreio-encomendas).
     * Application URL: O endereço do site (ex: meudominio.com/rastreio).
     * Application startup file: Digite 'index.js'.
   - Clique em "CREATE".

3. Instalar Dependências
   - Após criar, o cPanel mostrará um botão "Run NPM Install". Clique nele.
   - Aguarde a instalação finalizar (pode levar alguns minutos).

4. Banco de Dados e Variáveis de Ambiente
   - O arquivo '.env' já está incluído com as configurações básicas.
   - O banco de dados está configurado como SQLite (arquivo local 'prisma/dev.db'), então não é necessário criar um banco MySQL manualmente agora. O arquivo do banco já vai junto no zip.
   - Se precisar alterar configurações (como chaves de API do Resend ou WayMB), edite o arquivo '.env' pelo Gerenciador de Arquivos.

5. Acessar
   - Clique em "Restart" na aplicação Node.js.
   - Acesse a URL configurada.

IMPORTANTE:
- Se der erro 500 ou 404, verifique os logs no cPanel ou no arquivo 'stderr.log' na pasta da aplicação.
- Certifique-se de que a porta não está fixa no código (o sistema já usa `process.env.PORT` que é o correto para cPanel).

DÚVIDAS FREQUENTES:
- "Posso só jogar os arquivos lá igual PHP?"
  NÃO. O Node.js precisa estar rodando como um serviço. Por isso é necessário usar a ferramenta "Setup Node.js App".
